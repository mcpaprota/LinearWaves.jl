LinearWaves.LinearWaves — Module

LinearWaves is an implementation of a linear water wave theory.

LinearWaves.dispersion_relation — Method
dispertion_relation(d, ω; g=LinearWaves.g, ϵ=LinearWaves.tol)

Calculate a wave number k based on depth d, an angular wave frequency ω and gravitational acceleration g for a given accuracy ϵ according to linear wave theory."
